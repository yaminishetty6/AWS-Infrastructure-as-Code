# AWS-Infrastructure-as-Code
This project demonstrates how to provision AWS infrastructure
Instead of manually creating resources in the AWS Management Console, Terraform automates the entire infrastructure deployment, making it reproducible, scalable, and version-controlled.
The infrastructure created includes:
         ✅ Amazon VPC
         ✅ Public Subnet
         ✅ Internet Gateway
         ✅ Route Table
         ✅ Route Table Association
         ✅ Security Group
         ✅ EC2 Instance
Learn Infrastructure as Code (IaC)
- Automate AWS resource provisioning
- Build reusable Terraform configurations
- Understand AWS networking concepts
- Deploy an EC2 instance with a web server
- Version infrastructure using Git & GitHub
